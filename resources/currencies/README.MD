# Currencies

This directory holds the seeding information for the default currencies in the database.

For `fiat` currencies the codes are complying with the ISO 4217.

For `crypto`, we are using the most common codes instead. e.g `XBT -> BTC`

> *The ISO code for Bitcoin is XBT -- [the 'X' prefix is reserved for non-country currencies such as precious metals and supranationals](https://en.wikipedia.org/wiki/ISO_4217#X_currencies_(funds,_precious_metals,_supranationals,_other))*.

The `currencies.json` was based on [currencies.json](https://github.com/ourworldincode/currency/blob/main/currencies.json). See atribuition below.
